查询硬盘列表
```shell
fdisk -l
```

运行分区工具
```shell
fdisk /dev/sdb
# n 创建新分区
# w 保存并退出
````

创建文件系统
```shell
mkfs.ext4 /dev/sdb1  
```


临时挂载
```shell
mount /dev/sdb1 /mnt/new_disk
```

永久挂载
```shell
# 查询分区id
blkid /dev/sdb1

# 编辑配置文件
nano /etc/fstab
# 添加如下配置
# UUID=xxx /mnt/new_disk ext4 defaults 0 2

查看挂载状态
```shell
df -h
```


迁移数据
```shell
rsync -av /data/ /mnt/new_disk/data/
```
